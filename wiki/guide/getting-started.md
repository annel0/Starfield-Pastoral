# 快速开始

欢迎来到 **StardewCraft** — 一个在 Minecraft 中重现 Stardew Valley 的模组。

## 安装要求

| 项目 | 版本 |
|------|------|
| Minecraft | 1.21.1 |
| NeoForge | 21.1.217+ |
| Java | 21+ |

## 安装步骤

1. 安装 [NeoForge](https://neoforged.net/) 对应版本
2. 下载 StardewCraft 最新 `.jar` 文件
3. 将 `.jar` 放入 `.minecraft/mods/` 目录
4. 启动游戏，选择 NeoForge 配置

<InfoBox type="tip" title="兼容性提示">
推荐同时安装 Jade 模组以获得作物生长进度提示。
</InfoBox>

## 快速体验

进入世界后，你可以：

- **种植作物** — 用锄头右键草地开垦，种下种子并每天浇水
- **进入矿洞** — 找到矿洞入口，开始探索 120 层地下世界
- **钓鱼** — 制作鱼竿，在水边开始钓鱼小游戏
- **与 NPC 交流** — 在村庄寻找星露谷的经典角色
