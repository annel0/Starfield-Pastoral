# 作物系统

StardewCraft 完整还原了星露谷物语中的 **39 种作物**，包括生长阶段、季节限制、再生机制和肥料加速。

## 核心机制

### 生长阶段

每种作物有 **4 个生长阶段**（阶段 0 → 3），每阶段需要特定天数。在最后一个阶段完成后即可收获。

<InfoBox type="tip" title="Speed-Gro 加速">
使用肥料可以显著缩短生长时间。Speed-Gro 减少 10%，Deluxe 减少 25%，Hyper 减少 33%。
</InfoBox>

### 再生作物

部分作物收获后 **不会消失**，而是退回到上一阶段并重新生长。这类作物是长期收益的核心。

### 品质系统

收获物有 4 种品质：<span class="sc-badge sc-badge-green">普通</span> <span class="sc-badge sc-badge-blue">银星</span> <span class="sc-badge sc-badge-purple">金星</span> <span class="sc-badge sc-badge-amber">铱星</span>

品质越高售价越高。肥料和耕种技能会影响收获品质概率。

## 肥料效果一览

| 肥料 | 效果 | 适用场景 |
|------|------|---------|
| Speed-Gro | 减少 10% 生长时间 | 常规加速 |
| Deluxe Speed-Gro | 减少 25% 生长时间 | 长周期作物 |
| Hyper Speed-Gro | 减少 33% 生长时间 | 最大化产量 |
| Basic Fertilizer | 提高品质概率 | 提升银星概率 |
| Quality Fertilizer | 显著提高品质 | 提升金星概率 |
| Deluxe Fertilizer | 大幅提高品质 | 铱星概率最大化 |
