# 小桶物品注册规划（草案）

> 目标：为“小桶”准备完整的输入/输出物品注册清单与分类规则。当前结论：蜂蜜要做；茶叶/大米及其产物暂不做。
> 注意：果酒/果汁/饮品的**价格与可食用效果均为逐项不同**，不能用统一公式代替，必须按 Wiki/原版数据逐项注册。

---

## 1. 基本规则（对齐 Wiki）

- **水果 → 果酒**（产物与原料一一对应，贴图与物品 ID 需区分；每个果酒价格不同）
- **蔬菜/可食用采集物（不含蘑菇） → 果汁**（同样一一对应；每个果汁价格不同）
- **特殊配方**
  - 小麦 → 啤酒
  - 啤酒花 → 淡啤酒
  - 蜂蜜 → 蜂蜜酒（需要先新增“蜂蜜”）
  - 咖啡豆×5 → 咖啡
  - 茶叶 → 绿茶（暂不做）
  - 大米×1 → 醋（暂不做）

---

## 2. 需要新增/确认的基础原料

- **蜂蜜（必须新增）**
  - 物品注册 + 贴图 + 语言
  - 作为小桶输入产生“蜂蜜酒”

- **茶叶（暂不做）**
- **大米（暂不做）**

## 4. 需要的分类/标签支持

为保证“小桶”规则稳定，建议准备以下分类能力：

- `stardewcraft:fruit`（水果）
- `stardewcraft:vegetable`（蔬菜）
- `stardewcraft:edible_forage_no_mushroom`（可食用采集物，排除蘑菇）
