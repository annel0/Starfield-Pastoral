### 四、危险模式加成 (Danger in the Deep)

当危险矿井/骷髅洞穴入侵激活时：

```
伤害 = 基础伤害 × (1 + 难度 × 0.25)
最小伤害 = 20 + (难度-1) × 20

生命 = 基础生命 × (1 + 难度 × 0.5)
最小生命 = 500 + (难度-1) × 300

韧性 += 难度 × 韧性
```

### 五、防御计算公式

```
实际受到伤害 = max(1, (原始伤害 ± 12.5%随机) - 防御值)

// 当防御值 ≥ 伤害的50%时
防御值 = 防御值 × (1 - random(0, 0.2))
```

## 🎮 双伤害系统设计方案

### 核心理念

你的想法非常好！设计两套伤害系统：
1. **星露谷维度内**: 保持星露谷原汁原味的战斗体验
2. **主世界/其他维度**: 映射为更贴近原版MC的伤害系统

### 伤害映射比例分析

#### 星露谷数值特点
- **玩家HP**: 100-205
- **武器伤害**: 2-120 (低级2-5, 高级60-120)
- **怪物伤害**: 5-35 (普通5-23, 精英30-35)
- **怪物HP**: 24-1000+ (普通24-410, 精英500-1230)

#### Minecraft原版数值特点
- **玩家HP**: 20 (10❤)
- **武器伤害**: 4-8 (铁剑6, 钻石剑7, 下界合金剑8)
- **怪物伤害**: 2-15 (僵尸2-4, 凋灵骷髅5-10, 末影龙6-15)
- **怪物HP**: 10-200 (僵尸20, 凋灵300, 末影龙200)

### 推荐映射方案

#### 方案A: 5:1 线性映射 (推荐)

```
星露谷伤害/生命 ÷ 5 = MC伤害/生命

优点: 简单直观, 保持相对比例
缺点: 高级内容可能偏强
```

| 星露谷数值 | MC映射数值 | 说明 |
|------------|------------|------|
| 100 HP | 20 HP (10❤) | 初始生命 |
| 205 HP | 41 HP (20.5❤) | 满级生命 |
| 5 伤害 | 1 伤害 | 绿色史莱姆 |
| 23 伤害 | 4.6 伤害 | 飞蛇 |

| 100-120 伤害 | 20-24 伤害 | 无限木槌 |

#### 方案B: 分段非线性映射

```javascript
// 低伤害区 (1-20): 接近1:1
// 中伤害区 (21-50): 2:1
// 高伤害区 (51-100): 3:1
// 超高伤害区 (100+): 4:1

function mapDamage(sdvDamage) {
    if (sdvDamage <= 20) return sdvDamage * 0.5;      // 1-10
    if (sdvDamage <= 50) return 10 + (sdvDamage - 20) * 0.33; // 10-20
    if (sdvDamage <= 100) return 20 + (sdvDamage - 50) * 0.2;  // 20-30
    return 30 + (sdvDamage - 100) * 0.1;              // 30+
}
```

### 伤害计算伪代码

```python
def calculate_damage(attacker, target, weapon, dimension):
    # 获取基础伤害
    if dimension.is_stardew():
        base_damage = random(weapon.damage_min, weapon.damage_max)
    else:
        base_damage = random(weapon.mapped_damage_min, weapon.mapped_damage_max)
    
    # 暴击检测
    if random() < weapon.crit_chance:
        base_damage *= weapon.crit_multiplier
        if attacker.has_profession("desperado"):
            base_damage *= 2
    
    # 职业加成
    if attacker.has_profession("fighter"):
        base_damage *= 1.10
    if attacker.has_profession("brute"):
        base_damage *= 1.15
    
    # 防御计算
    defense = target.defense
    if defense >= base_damage * 0.5:
        defense *= random(0.8, 1.0)  # 防御衰减
    
    final_damage = max(1, base_damage - defense)
    
    # 应用随机波动 ±12.5%
    final_damage *= random(0.875, 1.125)
    
    return int(final_damage)
```

## 📝 注意事项

1. **玩家预期管理**: 在星露谷维度内应保持原汁原味的体验，数值不应变化太大
2. **平衡性**: 出维度后的伤害不应过于碾压原版内容，但可以略强
3. **一致性**: 同一武器在不同维度的相对强弱顺序应保持一致
4. **反馈**: 玩家应能清楚知道当前使用的是哪套伤害系统
