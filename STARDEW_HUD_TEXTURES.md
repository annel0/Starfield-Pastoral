# 星露谷玩家HUD材质规格

## 文件位置
```
src/main/resources/assets/stardewcraft/textures/gui/
├── stardew_bars.png     (256x64, 条形图)
└── stardew_hud_icons.png (64x32, 图标)
```

## 1. stardew_bars.png (256x64)

### 布局
```
┌────────────────────────────┐
│ 生命条背景 (0,0, 81x9)     │  <- 暗灰色边框
│ 生命条填充 (0,9, 81x9)     │  <- 红色渐变
│ 能量条背景 (0,18, 81x9)    │  <- 暗灰色边框
│ 能量条填充 (0,27, 81x9)    │  <- 青绿色渐变
└────────────────────────────┘
```

### 颜色建议
- **生命条背景**: `#3A3A3A` (深灰) + `#1A1A1A` (边框)
- **生命条填充**: `#E74C3C` (红色, 会被代码渲染时颜色覆盖)
- **能量条背景**: `#3A3A3A` (深灰) + `#1A1A1A` (边框)
- **能量条填充**: `#27AE60` (绿色, 会被代码渲染时颜色覆盖)

### 绘制规格
- 宽度: 81像素（对应MC原版9格血条）
- 高度: 9像素（和MC原版一致）
- 建议绘制时添加1像素的浅色高光（顶部）和深色阴影（底部）

## 2. stardew_hud_icons.png (64x32)

### 布局
```
┌─────┬─────┬─────┬─────┐
│ ⚡  │ ❤️  │ 💀  │ 😴  │
│(0,0)│(9,0)│(18,0)│(48,0)│
│ 9x9 │ 9x9 │ 9x9 │ 9x9 │
└─────┴─────┴─────┴─────┘

当前使用：
- (16,0): 能量图标 - 闪电/星星图标
- (0,0):  生命图标 - 心形图标
- (48,0): 疲惫图标 - 困倦/ZZZ图标
```

### 绘制规格
- 图标大小: 9x9像素
- 风格: 星露谷物语像素风格
- 建议添加1像素描边以提高可读性

### 图标设计参考
1. **能量图标** (16,0)
   - 闪电形状或星星形状
   - 青绿色系 `#3498DB`
   - 可以参考星露谷原版能量条的图标

2. **生命图标** (0,0)
   - 心形
   - 红色系 `#E74C3C`
   - 可以参考MC原版心形，但改成星露谷像素风格

3. **疲惫图标** (48,0)
   - ZZZ或困倦表情
   - 灰色系 `#95A5A6`
   - 显示在能量条左上角

## 材质绘制工具推荐
- Aseprite (推荐)
- GraphicsGale
- Pixelart 在线编辑器

## 导出设置
- 格式: PNG
- 不要使用压缩
- 确保透明通道正确
- 像素完美对齐（不要有半透明边缘）

## 代码整合说明

材质文件放置后，游戏会自动加载。HUD会在以下条件下显示：

1. ✅ 非创造/旁观模式
2. ✅ 在星露谷物语维度
3. ✅ 能量条常驻显示在左下角（物品栏左上方）
4. ✅ 生命条条件显示在右下角（生命值<最大值时）

### 渲染效果
- 条形图会根据当前值百分比填充（红→黄→绿渐变）
- 受伤时生命条会抖动
- 疲惫时能量条会抖动并显示困倦图标
- 数值文本显示在条的右侧，带黑色描边
- 低血量时闪烁粉色效果
