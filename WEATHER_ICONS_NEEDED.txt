=== 命令使用说明 ===
所有天气和时间命令已整合到主 /stardew 命令下：

时间管理:
- /stardew time get - 查询当前时间
- /stardew time set time <600-1560> - 设置当天时间
- /stardew time set date <1-28> - 设置日期
- /stardew time set season <spring|summer|fall|winter> - 设置季节
- /stardew time set year <1+> - 设置年份
- /stardew time add <分钟> - 增加时间
- /stardew time newday - 进入新的一天
- /stardew time reset - 重置到 Spring 1, Year 1, 6:00 AM

天气管理:
- /stardew weather get - 查询当前天气
- /stardew weather set <sun|rain|storm|snow|windspring|windfall|festival> - 设置天气
- /stardew weather tomorrow - 查看明天天气预测
- /stardew weather test - 测试天气概率分布（模拟100天）

注: 旧的 /stardewtime 和 /stardewweather 命令已被移除
